# Car Price Prediction Project

This is a Machine Learning web application that predicts the price of a car based on various features using a trained model.

## 🚀 Features
* Web Interface: Built with Streamlit for a user-friendly experience.
* Model: Uses a trained regression model (CPQ.pkl).
* Real-time Prediction: Enter car details and get an instant price estimate.

## 🛠️ Tech Stack
* Language: Python
* Libraries: Pandas, NumPy, Scikit-learn, Streamlit
* Deployment: GitHub & Streamlit Cloud

## 📂 Project Structure
* app.py: The main script for the Streamlit web interface.
* CPQ.pkl: The trained machine learning model.
* cleaned_data: The dataset used for the project.
* requirements.txt: List of Python libraries needed to run the app.
